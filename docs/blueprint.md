# **App Name**: QuickMoM

## Core Features:

- Meeting Creation: Allow users to create new meetings with titles, descriptions, and attendees.
- Topic Management: Enable users to add topics to meetings to categorize discussion points.
- Discussion Point Capture: Allow users to add and organize discussion points under specific topics.
- Action Item Assignment: Enable users to assign action items to meeting attendees with deadlines.
- Meeting Summary (AI): Generate automated summaries of meeting content using OpenAI to extract key points.
- Action Item Extraction (AI): Extract and list structured action items from meeting discussions using OpenAI as a tool.
- Speech-to-Text Transcription (AI): Convert audio recordings of meetings into text using OpenAI Whisper, displayed using the UI.

## Style Guidelines:

- Primary color: Soft teal (#64CCC9) for a calm, productive environment.
- Background color: Light, desaturated teal (#D5E8E8).
- Accent color: Warm coral (#F78888) for highlighting key actions and calls to action.
- Body and headline font: 'PT Sans' for readability and a modern, slightly warm feel.
- Use flat, minimalist icons for topics, action items, and AI functions to ensure clarity.
- Use a clean, card-based layout to display meetings, topics, and points. Prioritize key actions like 'Add Topic' and 'Summarize'.
- Use subtle transitions and loading animations for AI features to provide feedback without overwhelming the user.
